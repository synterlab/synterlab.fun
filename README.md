# SYNTERLAB - synterlab.fun

Premium audio plugin landing page for DAW producers.

## Sections
- Hero with floating particles (Web Audio API canvas)
- Marquee scrolling ticker (plugin names + DAW names)
- Featured plugin: SynTuner Pro with live Web Audio tuner demo
- Full plugin grid (6 plugins)
- Pricing (Free / Essential 9 / Studio 9)
- DAW compatibility
- Testimonials
- Footer

## Tech
- Single-file HTML with embedded CSS and JS
- Google Fonts: Space Grotesk
- Web Audio API (pitch detection via autocorrelation)
- No frameworks, no dependencies
- Mobile responsive

## Deploy
Open index.html in any browser. Works offline except Google Fonts.
