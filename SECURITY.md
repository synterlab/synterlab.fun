# Security Policy

## Scope

This security policy applies to the SYNTERLAB landing page hosted at [synterlab.space](https://synterlab.space).

## Reporting a Vulnerability

If you discover a security issue (XSS, content injection, clickjacking, etc.) please report it responsibly.

**Do not** open a public GitHub issue for security vulnerabilities.

**Contact**: security@synterlab.space

Include in your report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Your suggested fix (optional)

We will acknowledge your report within 48 hours and aim to release a fix within 7 days for critical issues.

## Supported Versions

Only the current live version at [synterlab.space](https://synterlab.space) is supported.
